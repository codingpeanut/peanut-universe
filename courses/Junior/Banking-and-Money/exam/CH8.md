**1. 下列有關貨幣基數、貨幣乘數與貨幣供給額的敘述，何者是錯誤的？**

正確答案: **B. 貼現放款是中央銀行可以完全控制的**

**計算過程與解釋:**

- 貼現放款是中央銀行對商業銀行的貸款，但並非中央銀行可以完全控制。它取決於商業銀行的需求和對貸款的依賴程度。

---

**2. 在其他情況不變下，中央銀行的資產增加，會使準備貨幣：**

**9. 假設中央銀行欲使貨幣供給額從9,000億增加至10,500億，在貨幣乘數為3的情況下，貨幣基數必須：**

正確答案: **C. 從3,000億增加到3,500億**

**計算過程與解釋:**

- 計算貨幣供給額變動：
    - 初始貨幣供給額 = 9,000億
    - 目標貨幣供給額 = 10,500億
    - 貨幣乘數 = 3
    - 增加的貨幣供給額 = 10,500億 - 9,000億 = 1,500億
    - 增加的貨幣基數 = 1,500億 ÷ 3 = 500億
    - 所以貨幣基數應從3,000億增加到3,500億。

**10. 假設中央銀行欲使貨幣供給額從10,000億增加到12,000億，在貨幣乘數維持2不變的情況下，央行進行公開市場操作必須買入多少金額的債券？**

正確答案: **E. 1,000億**

**計算過程與解釋:**

- 計算貨幣供給額變動：
    - 初始貨幣供給額 = 10,000億
    - 目標貨幣供給額 = 12,000億
    - 增加的貨幣供給額 = 12,000億 - 10,000億 = 2,000億

**12. 假設銀行僅吸收活期存款，活期存款的法定準備率為10%，通貨淨額對活期存款比率為10%，超額準備對活期存款比率為5%，則貨幣乘數為：**

正確答案: **C. 4.4**

**計算過程與解釋:**

- 法定準備率 = 10% = 0.1
- 通貨淨額對活期存款比率 = 10% = 0.1
- 超額準備對活期存款比率 = 5% = 0.05
- 貨幣乘數公式為： $$m = \frac{(1 + \text{通貨比率})}{(\text{法定準備率} + \text{通貨比率} + \text{超額準備比率})}$$
    - $$m = \frac{(1 + 0.1)}{(0.1 + 0.1 + 0.05)} = \frac{1.1}{0.25} = 4.4$$
    - 所以貨幣乘數是4.4。

**19. 假設：活期存款的法定準備率=10%，通貨淨額=6,000億，活期存款=10,000億，超額準備=100億，則：**

正確答案: **C. 貨幣供給乘數=2.25**

**計算過程與解釋:**

- 準備貨幣 = 通貨淨額 + 超額準備 = 6,000 + 100 = 6,100億
- 貨幣供給額M1A = 通貨淨額 + 活期存款 = 6,000 + 10,000 = 16,000億
- 貨幣乘數 = $$\frac{\text{準備貨幣}}{\text{法定準備金}} = \frac{6,100}{10,000 \times 0.1} = 2.25$$
    - 所以貨幣乘數是2.25。

**24. 假設法定準備率=0.25，通貨淨額=400，超額準備為零，支票存款1200，則貨幣供給乘數大約是多少？**

正確答案: **C. 2.3**

**計算過程與解釋:**

- 法定準備率 = 0.25
- 超額準備比率 = 0
- 通貨淨額 = 400
- 支票存款 = 1200
- 貨幣乘數公式為： $$m = \frac{1 + \frac{400}{1200}}{0.25 + \frac{400}{1200} + 0} = \frac{1.333}{0.583} = 2.3$$
    - 所以貨幣乘數是2.3。
